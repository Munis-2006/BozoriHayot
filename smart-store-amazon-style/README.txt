БОЗОРИ ҲАЁТ — ВЕРСИЯИ МУТОБИҚШУДА БАРОИ ТОҶИКИСТОН

Файлҳо:
- index.html
- styles.css
- script.js

Тарзи кор:
1. Папкаро кушоед.
2. Файли index.html-ро дар браузер боз кунед.
3. Ягон npm, сервер ё build лозим нест.

Чӣ тағйир дода шуд:
- Интерфейс ба забони тоҷикӣ мутобиқ карда шуд
- Нархҳо бо сомонӣ нишон дода мешаванд
- Матн ва пешниҳодҳо барои харидори Тоҷикистон наздиктар шуданд
- Қисмҳои расондан, пардохт, кафолат ва гирифтани мустақил илова шуданд
- Тарроҳӣ гармтар, оилавитар ва эътимодбахштар шуд

Эзоҳ:
Ин лоиҳа намунаи статикӣ барои намоиши дизайн мебошад.
